# 🎟️ Croatian Football League Ticket Seller

Static web application that simulates **buying tickets for Croatian First Football League matches**.

The project was developed as a **college project** and demonstrates frontend functionality such as dynamic data loading, custom UI components, and client-side data handling using JavaScript and JSON.

Instead of completing a purchase, the website **redirects users to official club websites** for real ticket sales.

---

## 🌐 Live Demo

🔗 [Vercel](https://ticket-seller.vercel.app/)

---

## ✨ Features

- ⚽ Browse Croatian First Football League matches
- 📅 **Custom-built JavaScript datepicker** for selecting match dates
- 🎟️ Ticket purchase simulation interface
- 🔗 Redirects users to official club websites for ticket purchases
- 📱 Responsive design optimized for mobile devices
- 🗂️ Match and ticket data stored in **JSON files**
- ⚡ Dynamic content rendering using **JavaScript**

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Data Handling**
- JSON (for match and ticket data)

**Deployment**
- Vercel

---

## ⚙️ How It Works

- Match and ticket information are stored in **JSON files**
- JavaScript dynamically loads the match data and displays it
- Users select match date using a **custom JavaScript datepicker**
- The interface simulates a ticket purchase process
- The final step redirects users to the **official club ticket websites**
